Introduction:

JavaScript is a High level interpreted Programming Language.

  • JavaScript was created in 10 day's in 1995 by Brendan Eich,when working at Netscape and Mozilla.
  • The orginal name of JavaScript was Mocha(this name is choosen by founder of Netscape).In september 1995,name changed to LiveScript.Again in december 1995,name changed to JavaScript.
  • JavaScript is used in Webpages to add functionality,validate forms,communicating with Server and read write HTML Elements.
  • JavaScript is a client side Scripting Language.
  • JavaScript is case sensitive Programming Language.
