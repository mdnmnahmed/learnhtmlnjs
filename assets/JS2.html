Variables

When you work with a programming language like C,C++,JavaScript etc. you frequently need to hold data for later use.Variables are use to store & hold any Data & Values.
The "var" & "let" keyword is use to declare Variables.

Example

  • var variableA=4;
    variableA has the value of 4.

  • let variableB=5;
    variableB has the value of 5.

  • var variableC;
    variableC=6;
    variableC has the value of 6.

If we Add variableA,variableB,variableC the ourput will be 15.

Comment

In JavaScript Comments can be used to explain JavaScript code, and to make it more readable.
We can use 2 types of comment in JavaScript:

1.Single Line Comment ( // )

Single Line Comment use to comment only a single line

2.MultiLine Comment (*/ ... */)

MultiLine Line Comment use to comment multiple lines
